Import Loan Applications

Import Instructions

Please follow these guidelines when importing loan applications:

  • Use CSV or Excel format (.csv, .xlsx)
  • Ensure all required fields are filled
  • Follow the date format: YYYY-MM-DD
  • Monetary values should be in numbers only (no currency symbols)

Required Fields

Field Name Description Format/Example
full_name Customer's full name John Doe
dob Date of Birth 1990-01-31
contact_number Contact phone number 9876543210
email Email address john@example.com
pan_card PAN Card number ABCDE1234F
aadhar_number Aadhar Card number 123456789012
address Full residential address 123 Main St, City, State
income_source Source of income Salary/Business/Self-employed
monthly_income Monthly income amount 50000
loan_type Type of loan personal/home/vehicle/education
loan_amount Requested loan amount 500000
loan_tenure Loan tenure in months 24
purpose Purpose of the loan Home renovation
dsa_code DSA Code (if applicable) DSA123
